Class LinqIterableBase<T, TLinq>

Abstract base class for LINQ iterables.

remarks

This abstract class serves as the generic base class for all LINQ iterables provided by this library. The {@linkcode LinqIterable} class just derives from this class and adds no further features. Other classes such as LinqElements inherit from this class and provide further methods. The generics-based pattern used in this case means that all filter methods, returning a TLinq, for example, will return instances of the subclasses.

Type parameters

  • T

    The type of the elements of the sequence.

  • TLinq: LinqIterableBase<T, TLinq>

    The type of the LINQ iterable returned by filters.
